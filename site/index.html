The SSR-First
Reactive Framework

CuboMX enhances your Server-Side Rendered applications with modern, declarative, and powerful reactivity, keeping your HTML clean and your JavaScript in charge.

Get Started View on GitHub

JavaScript is Home

Keep your logic in JavaScript, where it belongs. CuboMX keeps your HTML clean and focused on structure.

Declarative Binding

Use simple, explicit directives to bind your DOM to your state. No magic, no complex inline programs.

Predictable State

A single, global, flat API for all your reactive state. No more hunting for scope or context.

SSR-First Hydration

Effortlessly hydrate your application's state from server-rendered HTML with powerful directives.

script.js 
CuboMX.component('counter', {
  value: 0,
  increment() { this.value++ },
});
index.html 
<div mx-data="counter">
  <span mx-text="counter.value"></span>
  <button mx-on:click="counter.increment()">
    Increment
  </button>
</div>
